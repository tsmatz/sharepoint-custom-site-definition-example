このデモでは、カスタムのサイト定義に、

1. カスタムのワークフローフィーチャーも追加し、ドキュメントライブラリに関連付けています
2. カスタムのマスターページフィーチャーも追加し、サイトのマスターとして設定しています

SiteDefinition1 をリビルドして配置すると、[ワークフロー統合サイト] という名前でサイト定義がインストールされます.
配置に失敗する場合は、再度プロジェクトを開きなおしてから配置をおこなってください.
(DemoSPWorkflowLibrary1 のインストールは不要です.)

アンインストール時は、bin/Debug/ フォルダへ移動し、

setup.bat /uninstall

を実行します
